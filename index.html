01. ABOUT_ME

// WHO AM I?

Comecei a programar aos 13 anos por pura curiosidade. O interesse surgiu naturalmente através da tecnologia e dos jogos — universos que sempre me fascinaram.

Os primeiros passos foram páginas simples em HTML, depois a estética veio com o CSS, e a lógica tomou forma com o JavaScript. Cada linha de código aprendida abria uma nova janela de possibilidades.

Com o tempo, passei a construir sistemas mais complexos, interfaces modernas e a explorar o desenvolvimento de jogos próprios — combinando programação com criatividade de uma forma que só o game dev proporciona .

Hoje continuo evoluindo constantemente, buscando novas tecnologias e criando experiências cada vez melhores. Cada projeto é uma oportunidade de crescimento.
